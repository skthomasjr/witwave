Send an A2A JSON-RPC request to bob's nyx agent at http://localhost:8099/ using a fixed context ID of `continuity-session`. This establishes the session under a deterministic UUID so that 003b can resume it by name.

```
curl -s -X POST http://localhost:8099/ \
  -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":"003a","method":"message/send","params":{"message":{"role":"user","messageId":"msg-003a","contextId":"continuity-session","parts":[{"kind":"text","text":"Remember the word PINEAPPLE. Acknowledge with REMEMBERED."}]}}}'
```

The test passes if the response contains `REMEMBERED`.
The test fails if the response does not contain `REMEMBERED`, is empty, or the agent is unreachable.
