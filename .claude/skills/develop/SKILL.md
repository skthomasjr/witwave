---
name: develop
description: Continuous improvement loop — fix bugs and risks, evaluate for new ones, repeat until clean
---

Fix everything that can be fixed autonomously. For anything that requires human input, post a GitHub Issue comment and skip it — do not stop.

Steps:

1. Run `/work-skills`, then `/evaluate-skills`. If new skill issues were filed, repeat step 1.

2. Run `/work-bugs`, then `/evaluate-bugs`. If new bug issues were filed, repeat step 2.

3. Run `/work-risks`, then `/evaluate-risks`. If new risk issues were filed, repeat step 3.

4. Run `/evaluate-gaps`, then `/work-gaps`.

5. Run `/evaluate-features`, then `/work-features`.

6. Run `/work-bugs`, then `/evaluate-bugs`. If new bug issues were filed, repeat step 6.

7. Report a one-paragraph summary: issues worked and resolved per category, any blocked items, and done.
