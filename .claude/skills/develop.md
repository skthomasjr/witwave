# develop

This is an orchestrator skill. It runs the full autonomous development cycle by delegating each phase to the appropriate skill in sequence. It does not implement any phase directly.

## Instructions

Run each phase in order. Complete each phase fully before moving to the next.

**Phase 1: Find bugs.**

Find bugs across all components.

**Phase 2: Refine bugs.**

Refine bugs across all components, including cross-cutting bugs with no component assigned.

**Phase 3: Approve bugs.**

Approve bugs across all components, including cross-cutting bugs with no component assigned.

**Phase 4: Implement bugs.**

Implement all approved bugs across all components, including cross-cutting bugs with no component assigned.

**Phase 5: Find risks.**

Find risks across all components.

**Phase 6: Refine risks.**

Refine risks across all components, including cross-cutting risks with no component assigned.

**Phase 7: Approve risks.**

Approve risks across all components, including cross-cutting risks with no component assigned.

**Phase 8: Implement risks.**

Implement all approved risks across all components, including cross-cutting risks with no component assigned.

**Phase 9: Find gaps.**

Find gaps across all components.

**Phase 10: Refine gaps.**

Refine gaps across all components, including cross-cutting gaps with no component assigned.

**Phase 11: Approve gaps.**

Approve gaps across all components, including cross-cutting gaps with no component assigned.

**Phase 12: Implement gaps.**

Implement all approved gaps across all components, including cross-cutting gaps with no component assigned.

**Phase 13: Discover features.**

Derive features from all open requests marked as ready.

**Phase 14: Refine features.**

Refine all pending features.

**Phase 15: Approve features.**

Approve all pending features.

**Phase 16: Implement features.**

Implement all approved features.

**Phase 17: Refine the docs.**

Review and update the project documentation to reflect any changes made during this cycle.
